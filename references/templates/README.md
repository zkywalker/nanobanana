# BananaHub Built-in Starter Templates

This directory contains the small starter pack shipped with BananaHub Skill. It is intentionally not a full template gallery.

## What Belongs Here

- High-frequency workflows that work across Gemini / Nano Banana and OpenAI GPT Image
- Low-maintenance templates with no heavy sample assets
- Examples that validate core skill capabilities: generation, edit prompting, workflow activation, provider-specific prompt variants

## Current Starter Pack

| Template | Type | Use |
|---|---|---|
| `product-white-bg` | prompt | Clean ecommerce/catalog product shots |
| `info-diagram` | prompt | Short-label infographic and flowchart cards |
| `article-one-page-summary` | workflow | One-image visual summaries from articles or long notes |
| `repo-explainer-diagram` | workflow | Repo-aware architecture or onboarding diagrams |
| `background-replace-edit` | prompt | Background replacement edits while preserving the subject |

## What Should Move to the Hub

Large style packs, logo systems, character-consistency workflows, campaign galleries, samples, and fast-changing model experiments should live in official or community BananaHub template repositories and be installed on demand.
